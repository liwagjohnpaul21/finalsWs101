About the Project

17MerchShop is a fan-inspired online merchandise store dedicated to the globally recognized K-pop group Seventeen. This website was created as a school project to showcase a modern and visually appealing e-commerce layout using HTML, CSS, and Bootstrap. The platform features a variety of Seventeen-inspired products such as albums, clothing, photocards, and accessories. It is designed to provide an engaging and user-friendly browsing experience for fans, also known as CARATs. This project demonstrates key web development concepts including responsive design, multi-page navigation, internal linking, and proper file organization. Although the website is not intended for actual commercial use, it reflects the structure and design of a real online merchandise store. Through the development of 17MerchShop, the creator aims to combine creativity and technical skills in building a functional and aesthetically pleasing website

Objectives:

  • Create responsive design
  • Implement internal linking
  • Use Bootstrap components
